# Elizabeth River Water Simulation

This project models water flow in the Elizabeth River area using Blender as part of an urban flooding visualization workflow.

## Project Goal
To simulate water behavior in a 3D environment and support hydrodynamic visualization for urban flooding research.

## Tools Used
- Blender
- GIS / elevation data
- Unity (planned integration)
- VR visualization workflow

## Contents
- Blender scene file
- Terrain assets / heightmaps
- Preview renders
- Project documentation

## Overview
The scene represents the Elizabeth River environment and explores how water simulation can support better flood visualization in immersive systems.
